# e-plantShopping

Welcome to **e-plantShopping** – an elegant online plant shopping application designed to bring nature closer to you!

## Project Overview

**e-plantShopping** is a modern, user-friendly e-commerce platform for purchasing a wide variety of plants. Built with React and Redux, this application showcases an extensive collection of plants across multiple categories, each with detailed descriptions and pricing information.

### Paradise Nursery – Where Green Meets Serenity

The application is branded as **Paradise Nursery**, featuring a curated selection of high-quality plants including:

- **Air Purifying Plants** - Snake Plant, Spider Plant, Peace Lily, Boston Fern, Rubber Plant, Aloe Vera
- **Aromatic Fragrant Plants** - Lavender, Jasmine, Rosemary, Mint, Lemon Balm, Hyacinth
- **Insect Repellent Plants** - Oregano, Marigold, Geraniums, Basil, Lavender, Catnip
- **Medicinal Plants** - Aloe Vera, Echinacea, Peppermint, Lemon Balm, Chamomile, Calendula
- **Low Maintenance Plants** - ZZ Plant, Pothos, Snake Plant, Cast Iron Plant, Succulents, Aglaonema

## Features

- 🌱 **Extensive Plant Catalog** - Browse through 30+ plants organized by category
- 🛒 **Shopping Cart** - Add plants to cart and manage quantities
- 💚 **Responsive Design** - Optimized for desktop and mobile devices
- ✨ **Smooth Navigation** - Seamless transitions between home, product listing, and cart views
- 📱 **Modern UI** - Clean and intuitive user interface with green aesthetic

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Ahmedma1her/e-plantShopping.git
cd e-plantShopping
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

## Usage

1. **Browse Plants** - Click on the "Plants" link in the navigation bar to view all available plants
2. **View Details** - Each plant card displays the plant name, image, description, and price
3. **Add to Cart** - Click on items to add them to your shopping cart
4. **Manage Cart** - Click the cart icon to view, update quantities, and remove items
5. **Checkout** - Proceed to checkout when ready to purchase
